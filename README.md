# Solana Rust Developer Journey 🚀

This repository documents my hands-on experience and progression in Solana smart contract development using Rust and the Anchor framework.

---

## 🛠️ Tech Stack & Tools
* **Language:** Rust 🦀
* **Framework:** Anchor Framework ⚓
* **Blockchain:** Solana (Devnet / Localnet)
* **Environment:** WSL2 / Linux Terminal

---

## 📚 Core Concepts Covered
* **Anchor Smart Contracts:** Writing, testing, and deploying programs.
* **Account Management:** PDAs (Program Derived Addresses) & Account Validation.
* **Cross-Program Invocation (CPI):** Interacting with other Solana programs.
* **Security Practices:** Common vulnerabilities and contract optimization.
